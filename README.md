# MediPredic AI

AI-Based Disease Prediction Using Machine Learning

## Project Overview

MediPredic AI is a machine learning-based healthcare prediction system that predicts diabetes using patient medical data.

The project uses:
- Python
- Pandas
- Scikit-learn
- Logistic Regression
- Data Visualization

## Features

- Dataset loading
- Data cleaning
- Exploratory Data Analysis
- Machine Learning model training
- Accuracy evaluation
- Confusion matrix visualization
- Classification report
- Model saving/loading
- Real-time patient prediction

## Dataset

Dataset Used:
Pima Indians Diabetes Dataset

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Joblib
